# Project Title:AML e2e-eap-recon-new
Project Description: This project is designed for reconciling data between olympus and Mantas for AML (Anti-Money Laundering) Purpose. It invilves a multi-stage ETL ( Extract, Transform,Load)
process that includes data ingestion,transformation,reconcilation and reporting. The project leverages Hive for data warehousing , Spark for data Processing and Python for scripting and utility functions.
Features:Automated data ingestion and processing from Olympus and Mantas. Data quality checks at various stages of the ETL pipeline.
Reconcilation logic to identify matches, breaks, and exceptions. Generation of various report to anlaysis and investigation. Modular framework for handling different product processors. Configuratin-driven approach for flexibility and adaptability.
